"The Walls Will Fall: Joshua's Unstoppable Faith" (2025)

View Project →

A cinematic epic created using a 100% generative AI workflow to test the narrative limits of current video and audio generation models.

This experiment pushed modern AI tools to their storytelling edge, focusing on how visuals, sound, and editing combine when the entire pipeline is generated from high-level prompts.

  • Visuals: Generated with Veo 3 to create dramatic scenes and cinematic motion.
  • Audio: Composed with Suno AI for a fully automated soundtrack and sound design.
  • Assembly: Edited and finalized in Microsoft 365 Editor to produce the complete narrative.

The project tested how current AI tools interpret bold narrative structure and deliver a cohesive media experience from prompt to final cut.

View Project →

An exploration of sustained atmospheric and geological visualization using AI, testing how models handle prolonged physics-defying prompts.
